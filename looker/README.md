# Looker

Looker is a cloud-based business intelligence (BI) platform designed to explore and analyze data.

The docker image needed is built using https://github.com/honestica/docker-looker, you need a Looker license to build your own image.

Looker is run in standalone mode, you can configure an external database to make it stateless.
